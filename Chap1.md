> 2 types of common verbs in SV, verbs ends with **er** and **ar**

| AR-verb  |                     |
| -------- | ------------------- |
| to speak | att tala            |
| I speak  | jag tal**ar** |

| ER-verb        |                              |
| -------------- | ---------------------------- |
| to read        | att läsa                    |
| he/she/I reads | han/hon/jag läs**er** |

> Negate a sentence: add _**inte**_ after the verb.

| EN                      | SV                         |
| ----------------------- | -------------------------- |

| I speak Swedish.        | Jag talar svenska.         |
